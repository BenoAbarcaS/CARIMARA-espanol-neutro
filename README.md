# Traducción al Español Neutro para CARIMARA: Beneath the forlorn limbs

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Descripción

Esta es una traducción **NO oficial** y **NO automática** para el juego *CARIMARA: Beneath the forlorn limbs*. El objetivo es ofrecer un español mucho más **neutro y consistente**, corrigiendo los constantes cambios de vos, voceos mezclados y errores ortográficos presentes en la traducción oficial al español latinoamericano.

## ¿Qué corrige?

- Eliminación del voseo argentino ("vos", "sos", "tené", "podés")
- Uso consistente de "tú" y "usted" según el contexto
- Corrección de errores ortográficos
- Unificación de términos para que suenen más neutros
- Consistencia en el trato a lo largo de toda la historia
